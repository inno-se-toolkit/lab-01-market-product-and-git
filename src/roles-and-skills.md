# Roles and Skills Mapping

## 3.1 Roles per module
Based on the developed architecture diagram, the following technical roles are required to build and maintain the system:

| Module             | Roles involved                                            |
|--------------------|-----------------------------------------------------------|
| **Client-service** | Frontend Engineer (React/Next.js), UI/UX Designer, QA     |
| **API Gateway** | Backend Engineer, DevOps Engineer                         |
| **Moderation Layer**| Backend Engineer, ML Safety Researcher                   |
| **Inference Engine**| ML Engineer, Data Scientist, GPU Infrastructure Engineer |
| **NoSQL DB** | Database Administrator (DBA), Data Engineer               |
| **Monitoring** | SRE (Site Reliability Engineer), DevOps                   |

## 3.2 Cross-cutting skills
These are essential skills that almost every team member needs regardless of their specific role:
* **Git:** For version control and collaborative development.
* **Basic Linux usage:** Essential for navigating servers and managing GPU clusters.
* **Docker:** For containerizing services like the Inference Engine and API Gateway.
* **Network Fundamentals:** Understanding HTTPS/WSS protocols for secure data flow.
