# Tech Roles Mapping

This document maps the main components of Telegram to the engineering roles responsible for building and maintaining them.

---

## Frontend Engineer

**Main components:**

- Client Applications (Mobile / Desktop / Web)

**Responsibilities:**
Frontend engineers develop and maintain the user-facing applications. They are responsible for the UI, user interactions, performance on devices, and communication with backend APIs.

---

## Backend Engineer

**Main components:**

- API Gateway
- Messaging Service
- User & Authentication Service

**Responsibilities:**
Backend engineers design and implement core business logic, message processing, authentication, and API endpoints. They ensure scalability, reliability, and correct data handling.

---

## Storage / Backend Engineer

**Main components:**

- Media Storage Service

**Responsibilities:**
These engineers focus on storing and serving large media files efficiently, ensuring replication, fault tolerance, and fast access.

---

## DevOps / Site Reliability Engineer (SRE)

**Main components:**

- Deployment infrastructure
- Load balancers
- Cloud servers

**Responsibilities:**
DevOps engineers manage infrastructure, deployments, monitoring, and scaling. They ensure high availability and reliability of services.

---

## Security Engineer

**Main components:**

- Authentication Service
- Encryption mechanisms

**Responsibilities:**
Security engineers are responsible for protecting user data, implementing encryption, and securing authentication and communication channels.

---

## Product Manager (PM)

**Main components:**

- Overall system design
- Feature planning

**Responsibilities:**
Product managers define product requirements, prioritize features, and coordinate between teams to align technical work with user needs.

---

## Quality Assurance (QA) Engineer

**Main components:**

- Client Applications
- Backend APIs

**Responsibilities:**
QA engineers test features, identify bugs, and ensure the system works correctly across devices and services.
