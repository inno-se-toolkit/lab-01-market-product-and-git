## Roles and responsibilities

- **Mobile Engineer**
  - Makes the app for phones and fixes bugs.
  - Makes sure the interface works fast and doesn't lag.

- **Backend Engineer**
  - Makes the server part so it can handle many users.
  - Works on fast message delivery.

- **Cryptography Engineer**
  - Works with encryption so no one reads the chats.
  - Protects user data from hackers.

- **DevOps Engineer**
  - Sets up servers and makes sure they don't crash.
  - Automates stuff so everything works 24/7.

- **QA Engineer**
  - Searches for bugs in the app.
  - Checks if new features work on different phones.
