## Roles and responsibilities
1. Backend Engineer
   - API Development: Designs and implements server-side logic and maintains gRPC/RESTful APIs for service communication.
   - System Integration: Connects core services with databases and external providers like Yandex Pay.
   - Performance Tuning: Optimizes code to handle high-load traffic and ensures low-latency responses for real-time services.

2. SRE (Site Reliability Engineer)
   - Availability: Ensures the entire Yandex Go ecosystem stays online by setting up proactive monitoring and alerting.
   - Infrastructure as Code: Automates the deployment and scaling of microservices across different environments.
   - Incident Management: Leads the response to system failures and performs root-cause analysis to prevent future outages.

3. Data Scientist / ML Engineer
   - Algorithm Development: Builds and maintains the machine learning models that power the Pricing and Dispatch services.
   - Data Analysis: Analyzes historical trip data to improve routing efficiency and estimated arrival times.
   - Model Optimization: Continuously retrains models to adapt to changing city traffic patterns and demand spikes.

4. QA Automation Engineer
   - Test Automation: Writes scripts to automatically test the Mobile and Web applications across various devices.
   - Regression Testing: Ensures that new feature updates in Core Services do not break existing functionality.
   - Bug Tracking: Identifies, documents, and tracks software defects, working closely with developers to resolve them.

5. Database Administrator (DBA)
   - Cluster Management: Handles the installation, configuration, and maintenance of databases like YDB and ClickHouse.
   - Optimization: Monitors query performance and manages indexing to ensure fast data retrieval.
   - Security and Backup: Implements data security protocols and manages backup strategies to prevent data loss.
## Common skills across roles
1. Backend Engineer
   - Programming Languages: Expert knowledge of Go (Golang), C++, or Java.
   - API Protocols: Deep understanding of gRPC, Protobuf, and RESTful API design.
   - Database Proficiency: Experience with SQL/NoSQL and distributed transactions.
   - System Design: Knowledge of microservices architecture and high-load patterns.

2. SRE (Site Reliability Engineer)
   - Infrastructure as Code (IaC): Mastery of Terraform, Ansible, or Puppet.
   - Containerization: Proficiency in Docker and Kubernetes (K8s) orchestration.
   - Monitoring & Observability: Experience with Prometheus, Grafana, and ELK stack.
   - Scripting: Strong automation skills in Python, Bash, or Go.

3. Data Scientist / ML Engineer
   - Mathematical Foundations: Strong grasp of Linear Algebra, Calculus, and Statistics.
   - ML Frameworks: Proficiency in PyTorch, TensorFlow, or CatBoost (Yandex-native).
   - Data Processing: Experience with SQL, Python (Pandas/NumPy), and Spark.
   - Modeling: Expertise in Reinforcement Learning and Gradient Boosting.

4. QA Automation Engineer
   - Testing Frameworks: Experience with Appium, Selenium, or Pytest.
   - CI/CD: Understanding of Jenkins, GitLab CI, or TeamCity.
   - Mobile Debugging: Proficiency with Charles Proxy, ADB (Android), and Xcode tools.
   - Programming: Basic to intermediate skills in Python, Java, or Swift.

5. Database Administrator (DBA)
   - Distributed Systems: Expert knowledge of YDB (Yandex Database) or similar NewSQL.
   - Analytics Engines: Experience with ClickHouse and OLAP workload optimization.
   - Performance Tuning: Proficiency in query plan analysis and Linux kernel tuning.
   - Caching Systems: Deep understanding of Redis or Memcached internals.
